Unified AI Chatbot

This is a unified AI chatbot that integrates multiple knowledge sources using Streamlit, LangChain, FAISS, and Cohere. The chatbot supports queries across four domains: Wikipedia-based institution information, Indian Penal Code (IPC), Digital India, and Skill India documents.

 Features

- Institution Info Finder  
  Retrieve details like founder, year of establishment, and a summary from Wikipedia based on user input.

- IPC Chatbot  
  Ask legal questions related to the Indian Penal Code. The chatbot provides section-based responses from a preloaded document.

- Digital India Chatbot  
  Responds to queries using the official Digital India programme content.
